DIRECTIONS ON CHANGING YOUR WEB SITE TEXT

You must have HTML Web Publisher installed to use this function. Please contact info@lunarmedia.net if you would like pricing on installation.

You will be limited to the amount of changes you can apply to your web site text. Because this is using a script that will replace the text if the original is deleted, please be careful when you are updating your pages. We can not be held responsible for any deleted information data!

  1. Enter the password "admin" in the password box. Use lower case.
  2. Choose a file to edit from the drop down menu and click EDIT PAGE. If the file you want to edit is not listed, it may be a document such as a form that we do not recommend changing, or it has not been requested for addition. If you would like to add a new file, please contact info@lunarmedia.net

    BASIC HTML TAGS-----

  3. Regular text needs to be formatted or it will continue to run on. You can separate paragraphs by inserting a <P> tag between the paragraphs.
  4. To BOLD an item, you must surround the text with two tags. For example <B>WORD</B>
  5. To apply ITALICS you must surround the text with two tags. For example <I>WORD</I>
  6. To apply both bold and Italics, you can apply both tags around each other <B><I>WORD</I></B>
  7. To center text you can use <div align="center">WORD</div> tags around the word. You must CLOSE the tags to make the browser stop applying the effect you have commanded it to do.
  8. If you would like to shrink text, you must apply a set of tags such as <font size="-2">text</font>
  9. If you would like to enlarge text, you must apply a set of tags such as <font size="+1">text</font>
  10. If you would like to add a bullet before an item, use the <li>tags before the bulleted word
  11. If you would like to add an HTML link to an external web site, you can add it by surrounding the words you would like to be highlighted with the URL. For example: <a href="http://www.lunarmedia.net">Lunar Media Web Site</a>
  12. If you would like to add an image, we suggest contacting us before attempting to make such a drastic change to your layout. If you have already spoken with us and would like to insert a new image, please make sure your image is FTP'd to the server. There is a tutorial on FTP in your control panel.

    The proper code for adding images is <img src="images/imagename.gif" alt="" width="236" height="238" border="0"> If you do not know the width and height of your image, you may omit this from the code, however, it will not load as fast. The folder you put your images in is named images by default. If you have a different folder that the images are contained in, you would replace the word "images" with the title of that folder.

  13. If you have multiple areas of text in your site, they will be separated. This is to prevent any errors in the HTML code.
  14. Once you make all of your changes, click on Submit Changes
  15. Your page will refresh reflecting the new pages. You can click on the link above to view your page, or click LOG OUT, or MAIN MENU to return to the list of documents able to be edited.

NOTE: Be very careful in your selection, you can not undo once you click SUBMIT.
